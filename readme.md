## Usage ##

Create a new project base:

```bash
git clone https://github.com/stbsdk/boilerplate.git my-project
```

Remove git metadata, build and start services:

```bash
cd my-project
rm -rf .git
stbsdk
# or
DEBUG=* stbsdk
```
